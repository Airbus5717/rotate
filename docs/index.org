* Syntax Overview
** Basic variables
#+begin_src odin
io :: import("std/io");

// semicolons are optional
x: int = 1;
y := 2; // inferred
z :: 3; // z is const 3
d := ref x; // type is int*
d = nil; // now NULL

arr : []int = [1, 2, 3]; // inferred type ([3]int)
io.println(arr.len); // arr.len == 3
#+end_src

** Memory managment
similar to C, with some modern features (new/delete and defer)
#+begin_src odin
io :: import("std/io");
os :: import("std/os");

str := new [3]char;
defer delete str;

x := new [100]int;
tkn := new Token; // assuming Token is a struct
defer delete tkn;
defer delete x;

// how to check for failed allocation
if x == nil {
	io.println("Fail alloc");
	os.exit(1);
}
#+end_src
